Descrição:


O Microsoft Access é um Sistema de Gerenciamento de Banco de Dados (SGBD), ou seja, é um aplicativo utilizado na criação de Banco de Dados, nos quais devem ser guardados vários tipos de dados e informações. Ao final deste curso, o aluno desenvolverá uma agenda eletrônica utilizando as mais diversas técnicas para criação de um Banco de dados. OAccess pode ser utilizado em pequenos sistemas de cadastro de clientes estoques e orçamentos, entre outros.

Cursos desse pacote:


  • Lógica de programação
  • PHP
  • Java
  • Java Avançado
  • UML
  • Banco de dados
